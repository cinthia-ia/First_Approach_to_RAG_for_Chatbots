# Evaluation of Models Based on Retrieval-Augmented Generation for the Creation of Chatbots

The objective of this study is to conduct a comprehensive review of the existing literature on the topic of Retrieval-Augmented Generation (RAG) as it is applied to the creation of chatbots. Additionally, this study will evaluate five models for the information retrieval module of the RAG architecture through case studies in the context of a chatbot.

## Disclaimer

The original implementation was developed during a confidential industrial research project and therefore cannot be publicly released. This repository does not contain the original code or proprietary datasets. Instead, it documents the methodology, evaluation framework, and research findings in a way that preserves confidentiality while demonstrating the technical approach.
